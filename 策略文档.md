# BTC合约 趋势回调策略 v4.0

## 策略概要
- 版本: v4.0（替换v3.0模式B）
- 方向: LONG顺势追多 + SHORT顺势摸顶
- 仓位: 双向各1仓（做多做空可同时各持1仓）
- 每仓名义: 0.007 BTC
- 杠杆: 50x 逐仓
- 保证金/仓: ~10.7 USDT
- 止盈: +2.5%
- 止损: -1.5%
- 盈亏比: 1.67:1
- 扫描间隔: 5秒
- 多周期数据: 5m / 1h / 4h / 1d

## 入场条件（7条件全放宽）

### LONG 顺势追多
1. 4h close > SMA20（4h多头）
2. 1d close > SMA20（日线多头）
3. 价格在5m SMA20 ±1.0%内（回调中）
4. 1h ADX > 25（有趋势）
5. 4h ADX < 40（趋势不过热）
6. 放量条件：全放宽（无要求）
7. RSI(5m) > 40

### SHORT 顺势摸顶
1. 4h close < SMA20（4h空头）
2. 1d close < SMA20（日线空头）
3. 价格在5m SMA20 ±1.0%内
4. 1h ADX > 25
5. 4h ADX < 40
6. 放量条件：全放宽
7. RSI(5m) < 60

## 仓位规则
- LONG和SHORT各可持1仓
- 同向信号跳过（已有LONG仓则跳过新LONG信号）
- 反向信号不受影响（有LONG仓时可开SHORT）
