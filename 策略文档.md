## 一、核心指标说明

### 多空判断（每个周期独立计算）
- 多头: `price > MA7 且 MACD > MACD_Signal`
- 空头: `price < MA7 且 MACD < MACD_Signal`
- 周期: 1d、4h、1h、5m 各自独立计算，不跨周期比较

### 指标含义
| 指标 | 计算方式 | 含义 |
|------|---------|------|
| %b | (价格-布林下轨)/(布林上轨-布林下轨) | 价格在布林带相对位置，>0.85超买，<0.2超卖 |
| RSI | 相对强弱指数 | >70偏强，>82极强超买；<30偏弱，<40超卖 |
| ADX | 趋势强度指数 | ADX>25表示有趋势，ADX<25表示震荡 |
| 成交量比率 | 当前量/20根均量 | 放量>1.5x确认信号有效 |

**布林带参数:** 周期20，2倍标准差
**ADX参数:** 窗口14
**共用过滤:** 成交量 > 1.5x均值

### 【信号1】做多A - 逆势抄底

**性质:** 逆势，大周期空头 → 5m超卖反弹
**核心逻辑:** 大周期下跌中，价格急跌到布林下轨博弈反弹

**触发条件:**
1. 4h 均线空头（price < MA7 且 MACD < MACD_Signal）
2. 1d 均线空头（price < MA7 且 MACD < MACD_Signal）
3. 5m %b < 0.2 → 超卖
4. 5m RSI < 40 → RSI超卖确认
5. 4h ADX < 40 → 空头趋势未过强（避免在强空头中逆势）
6. 成交量 > 1.5x均值

**入场模板:**
```
【做多A】大周期空头+5m超卖反弹
入场: $价格
止损: $价格×0.97 (-3.0%)
止盈: $价格×1.05 (+5.0%) 全仓一次性
```

## 六、风控要点

1. 指标都是概率信号，非预测
2. 极端行情（瀑布/插针）指标可能失效
3. 20x杠杆波动大，止损必须严格执行
4. 成交量>1.5x是所有信号的共用过滤条件
5. 市价平仓时止盈止损自动撤销
6. 连续亏损3次后暂停交易30分钟
7. 平仓后冷却300秒才能再次开仓
8. 只挂2个条件单（止损+全仓止盈），逻辑简洁不易出bug
9. 单方向最多3仓
10. 补仓后撤销原SL/TP，以新均价重新挂单（v2.10）
11. 有持仓时继续监控，可追加开仓
