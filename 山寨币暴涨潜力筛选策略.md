# 山寨币暴涨潜力筛选策略

## 核心思想
找到**波动率压缩到极致** + **RSI处于低位** + **有放量启动信号**的币，等它突破。
核心逻辑：**横有多长，竖有多高**——布林带极度收口后，一旦放量就是行情启动。

---

## 筛选条件

### 必须全部满足

| 指标 | 条件 | 逻辑 |
|------|------|------|
| 布林带宽度 | 1d周期 BB宽度 < 8% | 极度收敛，蓄力 |
| RSI(1d) | < 55（最好 < 50） | 价格处于低位，上涨空间大 |
| 放量 | 近3根K线成交量 > 前20根均量 1.3x+ | 有资金进场 |
| 价格位置 | 处于近20日低点附近 | 不能追高 |

### 满足以下任意一项优先

- 4h K线突破布林上轨
- MACD 1d形成金叉
- 4h出现放量阳线

---

## 目标价计算

```
第一目标 = 布林上轨 + 半个布林范围
          （即突破后的理论涨幅）

止损 = 布林下轨下方 2-3%
```

---

## 风险控制

- **仓位：** 单币不超过总资金20%
- **止损：** 必设，控制在总资金 -2% 以内
- **不碰：** RSI>70的币（超买区域）
- **不碰：** 消息面有大利空的币（解锁/监管）

---

## 操作步骤

1. 每天收盘前扫一遍前50主流币
2. 筛选出 BB宽度 < 8% 的币（收敛形态）
3. 看 RSI(1d) < 50 的优先
4. 等放量突破布林上轨时买入
5. 挂止损 + 目标持3-7天

---

## 指标计算说明

### 布林带
- 周期：20
- 标准差倍数：2
- BB宽度 = (上轨 - 下轨) / 中轨 × 100%

### RSI
- 周期：14
- 使用1d周期

### 放量比
- 近期3根K线平均成交量 / 前20根K线平均成交量
