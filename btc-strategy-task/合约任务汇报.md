## 三、核心指标

| 指标 | 公式 | 作用 |
|------|------|------|
| %b | (价格-布林下轨)/(布林上轨-布林下轨) | 超买>0.90 / 超卖<0.15 |
| RSI | 相对强弱指数 | 超买>70 / 超卖<30 |
| MACD | 快线-慢线 | 金叉做多 / 死叉做空 |
| ATR | 真实波动幅度 | 辅助参考 |
| MA7 | 7周期均线 | 判断短期方向 |
| ADX | 趋势强度指数 | ADX>25有趋势，ADX<25震荡 |

参数设定：
- 布林带：周期20，2倍标准差
- MACD：快线12，慢线26，信号9
- ADX：窗口14

### 过滤1：成交量确认

## 十、执行流程（每轮10秒）

```
① 获取数据：5m + 1h + 4h + 1d K线
② 计算指标：MA/MACD/RSI/BB/ATR/%b/ADX/成交量
③ 风控过滤：放量>1.5x? 否 → 观望
④ 判断方向：4h+1d均线方向 + RSI区间
⑤ 逆势信号 → 额外检查 4h ADX<40? 否 → 观望
⑥ 有反对方向持仓 → 检查价格间隔≥1.5%? 否 → 观望
⑦ 检查信号：是否触发6种信号之一
⑧ 有信号 → 检查单方向仓位上限（≤3）→ 执行开仓 + 挂SL/TP + 推送微信
⑨ 无信号 → 继续观望
⑩ 循环等待10秒
```

## 十二、风控要点

1. 止损必须严格执行，20x杠杆波动大
2. 指标均为概率信号，非预测
3. 极端行情（瀑布/插针）指标可能失效
4. 成交量>1.5x过滤会减少假信号频率
5. 4h ADX>=40时逆势信号过滤，避免趋势过强时被碾压
6. 市价平仓时止盈止损自动撤销
7. 连续亏损3次后暂停交易30分钟
8. 止盈为一次性全仓，无分批
9. 单方向最多3个仓位（long/short各计各的）
10. 各仓位SL/TP独立，互不干扰
11. 震荡市（ADX<25）信号有效，不被过滤
12. **v2.9.2新增**：反向持仓价格间隔≥1.5%，允许对冲但需拉开距离
